Diese Datei wurde / wird dazu benutzt, zu testen wie das mit dem "livestream" hochladen so funktioniert.
Wir laden, wie bekannt sein sollte nämlich einfach eine Video Datei vom Bienenstock hoch. Diese Seite soll
dafür den Bienenstock demonstrieren und erlaubt uns eine Datei an den Server zu schicken um zu testen, ob
dieser die Datei richtig verarbeitet. Bitte nicht ohne Rücksprache ausprobieren, da dadurch Fehler
auftreten könnten.
